## Polls Project

This is a step by step follow up on Django documentation for a refresher on Django.

### Dependencies

1. virtualenv 20.24.2
2. python 3.11.4
3. Django 4.2.4
4. Django rest framework 3.14
5. pip 23.2.1

### Common Commands

- Running server locally: 

    `$ python manage.py runserver`

- Preparing database migrations:
    
    `$ python manage.py makemigrations <django app name>`

- Creating database entities migrations to the database:

    `$ python manage.py migrate`

- Creating Superuser/Admin:

    `$ python manage.py createsuperuser`

- Invoking the Python shell:

    `$ python manage.py shell`
